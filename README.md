**shot-url** est une solution open-source vous permettant de gerer avec sereniter vos liens *Linkedin*,*youtube* et autre sans stress.

##feature:

- racourcissement d'url
- generation de QRCODE

## setup

with pip

```bash

$ python -m venv .venv

$ . .venv/bin/activate

$ pip install -r requirement.txt

```
or with uv
